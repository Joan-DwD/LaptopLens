# LaptopLens

LaptopLens is a full stack web application that combines a laptop store with an AI powered chatbot guide, helping non-technical or technical users find the right laptop based on their needs and budget.

# User Stories

## User Roles

**Guest**: Unregistered visitor browsing the store

**Customer**: Registered user who can purchase laptops and save items

**Admin**: Store manager who manages products, orders, and dashboard


## Module 1: Authentication & Accounts

As a guest, I want to browse the store without creating an account so that I can view products.

As a guest, I want to register with email and password so that I can place orders.

As a customer, I want to log in and log out so that I can access my account functions.

As a customer, I want to reset my password if I forget it so that I can regain access.


## Module 2: Catalog & Product Pages

As a user, I want to view a catalog of available laptops so that I can browse options.

As a user, I want to view a dedicated product page for each laptop so that I can see full details.

As a user, I want to see whether a laptop is in stock so that I know if I can buy it.


## Module 3: Search & Basic Filtering

As a user, I want to search laptops by name or brand so that I can find a specific product quickly.

As a user, I want to filter results by price range and key specifications (for example RAM or storage) so that I can narrow down my choices.


## Module 4: Wishlist

As a customer, I want to save a laptop to my wishlist so that I can review it later.

As a customer, I want to view my wishlist so that I can review saved items.

As a customer, I want to remove items from my wishlist so I can keep it organized.


## Module 5: Comparison Tool

As a customer, I want to select multiple laptops so that I can compare them side by side.

As a customer, I want to see a side by side comparison view so that I can clearly see differences in specifications and price.


## Module 6: AI Chatbot Recommendation Engine

As a user, I want to open a chatbot so that I can get help choosing a laptop.

As a non-technical user, I want the chatbot to ask simple questions about my needs and budget so that it can understand my priorities.

As a user, I want the chatbot to recommend laptops based on my answers so that I can consider suitable options.

As a user, I want the chatbot to explain why it recommends certain laptops in plain language so that I understand the suggestions.

As a user, I want to restart or refine the chatbot conversation so that I can change my answers and get different results.

As a customer, I want to save chatbot recommendations to my profile so that I can review them later.


## Module 7: Cart & Checkout

As a customer, I want to add a laptop to my cart so that I can prepare to buy it.

As a customer, I want to view and edit my cart before checkout so that I can confirm items and quantities.

As a customer, I want to enter a shipping address during checkout so that the order can be delivered.

As a customer, I want to review an order summary before confirming the purchase so that I can verify the total cost.

As a customer, I want to choose a payment method and complete payment so that my order is paid for.

As a customer, I want to see confirmation when the payment succeeds so that I know the order was placed.

As a customer, I want to receive an error message when payment fails so that I can retry the payment.

As a customer, I want to view my order history so that I can check past purchases.

Payments should be handled using an external payment provider such as Stripe or PayPal rather than storing card information directly.


## Module 8: Admin Management

As an admin, I want to add new laptops to the catalog so that customers can buy them.

As an admin, I want to edit or remove laptops so that product information stays accurate.

As an admin, I want to view and update customer orders so that I can manage fulfillment.

As an admin, I want a dashboard showing key store metrics so that I can monitor store performance.


## Module 9: Notifications

As a customer, I want to receive an email confirmation when I place an order so that I have a record of my purchase.

As an admin, I want to be notified of critical problems such as repeated payment failures so that I can act quickly.

As a customer, I want to receive a notification when a laptop in my wishlist becomes available so that I know when I can purchase it.
